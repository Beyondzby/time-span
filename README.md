# 使用 github pages 和 issues 构建的事件计时小项目

用来管理每件事浪费了多少时间

## 项目展示

![主页](./homepage.png)

## 如何使用

### 一、创建自己的 repository

名称一定要是 `db`

可见性: private(别人看不到) 或者 public

### 二、生成自己的 access token

[github access token 生成网址](https://github.com/settings/tokens)

1. 点击 generate
2. 选择范围为 repo
3. 记住这个 access token

### 三、现在就可以登录了

username: 自己的 github 名称

password: 刚才生成的 Access Token
